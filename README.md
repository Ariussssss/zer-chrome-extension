# Zer-Chrome-Extension

A personal Chrome extension in development.

## Usage

### Quick locate

1. `z + tab`, turn on Zer extension prompt.
2. Type a `key + keywords` splited by space of `:`, for example `z zer chrome extension`

KeyList:

- b: bilibili
- ba: bilibili followings
- g: github
- s: stackoverflow
- y: YouTube
- z: zhihu
- Default: google
